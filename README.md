# dirty-dotfiles

## Setup

- Operation System: `manjaro`
- Window Manager: `i3-gaps`
- Shell: `bash`
- Display Manager: `lightdm`
- Terminal Emulator: `urxvt`
- File Manager: `ranger, pcmanfm`
- Application Launcher: `rofi`
- Compositor: `compton`
- Bar: `polybar`
- Notification: `dunst`
- Text Editor: `nano`
